## JRiver Media Center 升级是什么意思？

**JRiver Media Center（简称 JRMC / JMC）** 是一款发烧级多媒体播放与管理工具，支持 Windows、macOS 和 Linux 三平台，能处理音频（FLAC、DSD、APE 等无损格式）、视频（蓝光、4K）、图片，以及流媒体和家庭媒体网络。它的授权模式是**买断制**——购买一个版本后永久使用，子版本（如 35.0.1 → 35.0.42）免费更新，但如果想用下一个大版本（比如从 35 升到 36），就需要单独购买升级授权。

简单说：**子版本（小数点后的更新）免费；大版本跨版升级要另外付钱。**

## Win / Mac / Master 升级版有什么区别？

升级通道分三种规格，不是随便选的：

| 规格 | 覆盖平台 | 激活设备数 | 适合人群 |
|---|---|---|---|
| **Win 版授权升级** | Windows | 3 台 Win 设备 | 纯 Windows 用户 |
| **Mac 版授权升级** | macOS | 3 台 Mac 设备 | 纯 Mac 用户 |
| **Master 版授权升级** | Win + Mac + Linux | 10 台设备（跨平台）| 多平台用户或多台设备需求 |

## 升级后能用多久？更新政策是什么？

升级后的授权是**永久买断**，不是订阅制，不需要每年续费。

同一大版本内的小版本更新（比如 35.0.5 → 35.0.42）全部免费。但跨大版本（比如 35 → 36）需要再次付费升级。JRiver 的大版本发布周期通常在一年左右，不会很频繁。

换电脑或重装系统怎么处理？卸载原设备上的软件，再在新设备重新激活即可。兑换码可以重新生成授权文件，不会丢失使用权。

## 常见问题

**Q：我在其他平台买的 JRiver 授权，能通过数码荔枝升级吗？**

不一定。数码荔枝的升级通道是针对其销售渠道的授权设计的。如果你的授权是从 JRiver 官网或其他渠道购买的，建议购买前联系客服确认，避免买错。

**Q：授权文件过了 14 天失效了怎么办？**

不用担心，你的购买资格不会消失。用邮件里的兑换码，访问 JRiver 的临时授权兑换地址重新生成一个新文件就行。一年内可以生成 10 次，日常使用完全够。

**Q：Win 版授权能在 Mac 上用吗？**

不能。Win 版只限 Windows 设备，Mac 版只限 macOS 设备。需要跨平台使用，只能选 Master 版。

**Q：升级后还需要重新安装软件吗？**

不需要单独重装。已有的安装版本继续使用，打开升级授权文件激活后，软件会识别新的版本授权。如果想用最新的大版本，下载最新安装包覆盖安装即可，媒体库数据会保留。

**Q：我买了升级授权，以后还能再次升级吗？**

可以。下次有新大版本发布，重新购买升级通道即可，流程和这次完全一样。没有捆绑限制。
